¿Qué estructura de landing page queremos?

Tres enfoques distintos para comunicar Colmena en un pantallazo. Elegí uno como dirección base.

A

"The Dashboard" — Funcional de una pantalla

Hero con diagrama animado del ciclo multi-agente → 6 cards con capacidades clave → "How it works" visual → Quick start + CTA. El usuario entiende TODO sin scrollear más de 2 pantallas. Ideal para causar impresión inmediata en GitHub/HN.

B

"The Manifesto" — Narrativa + profundidad progresiva

Hero con el hexágono actual → manifiesto breve → "Why Colmena" (problema → solución) → features en detalle con capturas/diagramas → comparación con auto-mode → use cases → install. Viaje narrativo para el que quiere entender antes de confiar.

C

"The Bridge" — Híbrido con doble entrada

Hero impactante → 4 value props ultra-visuales (iconos + texto corto) → "How it works" en 3 pasos → Diagrama de arquitectura → Use cases → "Colmena vs Auto-Mode" → Install → CTA. Sirve tanto al que escanea rápido como al que lee todo. Requiere más scroll pero cubre todos los públicos.

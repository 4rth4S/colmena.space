View on GitHub
What Colmena gives you
I.
Deterministic governance. YAML rules, compiled regex, audit.log on every call. Zero LLM calls in the hot path, zero per-call cost.
II.
Multi-agent missions. Peer review with QPC scoring. ELO-calibrated trust per role, earned from observed behavior.
III.
Local-first. No cloud classifier. Every decision is explainable by the rule ID written to audit.log.
Quick start
$git clone git@github.com:4rth4s/colmena.git
$cd colmena && cargo build --workspace --release
$./target/release/colmena setup
